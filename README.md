# localWiki Community

Welcome to the official community hub for **[localWiki](https://localwiki.nextbyts.com)** — the local-first encrypted vault for dev team secrets and setup docs.

This repository is the place to:

- **Report bugs** you encounter in the desktop app
- **Request features** you'd like to see built
- **Give feedback** on the documentation
- **Ask questions** about how localWiki works

## Before you post

1. **Search existing issues** to avoid duplicates
2. **Read the [documentation](https://localwiki.nextbyts.com/docs)** — your answer might already be there
3. **Use the issue templates** — they help us respond faster

## Issue templates

| Template | When to use |
|----------|-------------|
| **Bug Report** | Something isn't working as expected |
| **Feature Request** | You have an idea for a new feature or improvement |
| **Documentation Feedback** | A docs page is wrong, confusing, or missing something |

## What this repo is NOT

This is **not** the source code repository for localWiki. The product source code is maintained privately. This repo exists solely for community communication.

## Response times

We're a small team. We read every issue and aim to respond within a few business days. Bug reports with clear reproduction steps get prioritized.

## Code of Conduct

Be respectful. Be constructive. We're all here to make localWiki better.

- No spam, self-promotion, or off-topic content
- No harassment, insults, or personal attacks
- Issues that violate these guidelines will be closed and the author may be blocked
